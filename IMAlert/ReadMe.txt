IM Alert Web Service Demo - Send Messages to MSN, Yahoo, ICQ, AIM

This code allows you to send instant message to four major providers MSN, Yahoo, ICQ, and AIM. You don’t even need an account or need to install any clients. It a unique web service project on PSC.
Note: this demo required PocketSOAP from www.pocketsoap.com (600 KB)
